How to use it?

  • You can create lists by clicking + New List
  • After you create a list, you can add specific tasks to it by clicking the + circle
  • If you want to change the task, you can do that by clicking on the edit icon
  • After you are done with the task, you can delete it by clicking the trash icon
  • You can edit or delete a list by clicking on the the gear icon
